    Shortscode

    Buttons:
    1. In the Post section, click on edit post or create a new post
    2. Click on HTML
    3. A code is written below use that code to display that Button link
    <!-- Default Button (Yellow) -->
<a class="btn " href="#your_link" target="_blank">Text</a>

    
<!-- Green Button -->
<a class="btn tw-green" href="#your_link" target="_blank">Text</a>

    
<!-- Red Button -->
<a class="btn tw-red" href="#your_link" target="_blank">Text</a>
